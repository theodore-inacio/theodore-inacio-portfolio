# 3. SQL Analytics & Market Research

This directory centralizes data analysis workflows utilizing relational databases to solve complex business and financial research questions.

### Key Technical Focus:
* **Advanced Querying**: Structuring multi-table operations with robust `JOIN` logic, subqueries, and Common Table Expressions (CTE).
* **Quantitative Analysis**: Leveraging Window Functions (`PARTITION BY`, `LAG/LEAD`) for price-action analysis, market trend identification, and customer behavioral segmentation (RFM).
