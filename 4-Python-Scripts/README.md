# 4. Python Scripts & Automated Workflows

This directory showcases tactical automation scripts designed to eliminate manual pipelines, extract unstructured data, and accelerate financial workflows.

### Key Technical Focus:
* **Financial Automation**: Scripts interacting with financial APIs (e.g., Yahoo Finance) to stream market metrics straight into analysis pipelines.
* **AI-Augmented Research**: Custom text-mining structures and advanced prompting logic to parse institutional reports and speed up qualitative research.
